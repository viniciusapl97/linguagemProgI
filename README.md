# Linguagem de Programação I

Repositório com as atividades desenvolvidas na disciplina de Linguagem de Programação I, organizadas por aulas e por linguagem de programação.


## Linguagens utilizadas

- Python
- Java
- C++

## Objetivo

O objetivo deste repositório é reunir os exercícios práticos da disciplina, aplicando conceitos fundamentais de programação, como:

- Variáveis e tipos de dados.
- Operadores aritméticos, relacionais e lógicos.
- Estruturas condicionais.
- Estruturas de repetição.
- Vetores, listas e estruturas de dados simples.
- Funções e modularização.
- Organização de código em múltiplas linguagens.

## Organização dos arquivos

Cada aula está separada em pastas próprias, com subpastas para cada linguagem:

- `python_logic`
- `java_logic`
- `cpp_logic`

Essa organização facilita a comparação entre sintaxe, lógica e estrutura em cada linguagem.

## Observações

- Os códigos foram comentados com foco didático.
- As implementações seguem a proposta dos roteiros de aula e listas de exercícios.
- Os arquivos foram separados de acordo com o enunciado de cada atividade.
